```
fdisk -l

fdisk /dev/vdb
输入n
输入p
...
输入w
完成分区工作

fdisk -l
这时候能够看到我们的分区了，接下来格式化分区
mkfs -t ext4 /dev/vdb1
在根目录下新建disk目录，然后将分区挂载到disk目录上
mount /dev/vdb1 /root/wwwroot
大功告成

配置开机自动挂载（查看信息）
sudo blkid
配置开机自动挂载（操作）
sudo vim /etc/fstab
加入：
UUID=xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxx /root/wwwroot ext4 defaults 0 1
修改完/etc/fstab文件后，运行
sudo mount -a
```
